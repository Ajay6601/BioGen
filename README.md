# BioGen

BioGen is a toolkit for generating and verifying biomedical analysis code from natural-language requests.

## Project Layout

- `biogen/`: Core package and pipelines
- `tests/`: Unit tests
- `data/`: Dataset setup notes

## Quickstart

```bash
python -m venv .venv
. .venv/Scripts/activate
pip install -r requirements.txt
python -m biogen.main --help
```
